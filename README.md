# End-to-End Nashville Real Estate Market Analysis

End-to-end real estate analytics project leveraging SQL and Tableau to analyze housing market trends, property sales performance, pricing patterns, and geographic variations across Nashville. The project transforms raw property transaction data into actionable business intelligence through data cleaning, exploratory analysis, and interactive dashboarding.

## Executive Summary

The Nashville housing market has experienced significant changes in property values, sales activity, and regional demand patterns. Understanding these trends is critical for real estate investors, developers, homeowners, and market analysts.

This project analyzes residential property transactions across Nashville to identify pricing trends, high-value locations, sales performance patterns, and market dynamics through SQL-based analysis and Tableau visualization.

## Key Business Insights

- Nashville recorded over 56,000 property transactions during the analysis period.
- Average property sale prices varied significantly across cities.
- Property values generally increased with the number of bedrooms.
- Certain cities consistently commanded premium market prices.
- Housing sales activity displayed seasonal fluctuations across months.
- A small number of locations generated substantially higher average sale prices than others.

## Business Problem

Real estate stakeholders require visibility into housing market performance to make informed investment and pricing decisions.

This project addresses key business questions including:

- Which cities command the highest property prices?
- How have housing prices evolved over time?
- Which property types generate the highest value?
- What trends exist in monthly property sales activity?
- How does property size influence market pricing?
- Which regions present premium real estate opportunities?

## Analytics Workflow

### Data Cleaning & Preparation (SQL)

Performed:

- Duplicate record removal
- Standardization of property addresses
- Null value handling
- Data type conversion
- Address parsing
- Data validation
- Dataset preparation for analysis

### Exploratory Data Analysis

Key questions answered:

- Which cities have the highest average sale prices?
- How do property prices vary by bedroom count?
- How have housing prices changed over time?
- What trends exist in monthly sales volume?
- Which locations represent premium housing markets?

### Dashboard Development (Tableau)

Created an interactive dashboard featuring:

- KPI Performance Overview
- Monthly Property Sales Trend
- Average Sale Price Trend
- Average Sale Price by City
- Average Sale Price by Bedrooms
- Dynamic Filtering by Year and City
